# 📚 Repositório de Estudos: IP & AED

Olá! 👋 Bem-vindo(a) ao meu repositório acadêmico. 

Este espaço foi criado com o objetivo de documentar, armazenar e organizar todas as questões, listas de exercícios e pequenos projetos que resolvi durante meus estudos universitários. 

Ele serve como um portfólio prático da minha evolução na lógica de programação e serve de base para a minha jornada e desenvolvimento na área de Engenharia de Software.

---

## 📂 O que você vai encontrar por aqui?

O repositório está dividido em duas categorias principais, correspondentes às disciplinas:

### 1. 🖥️ Introdução à Programação
* **Linguagem:** `C`
* **Foco:** Fundamentos da programação. 
* **Tópicos abordados:**
  * Lógica básica e estruturas de controle (`if/else`, `switch`).
  * Laços de repetição (`for`, `while`, `do-while`).
  * Manipulação de vetores (arrays) e matrizes.
  * Criação de funções e introdução ao uso de ponteiros.

### 2. 🧩 Algoritmos e Estruturas de Dados
* **Linguagem:** `C++`
* **Foco:** Organização, manipulação de dados e eficiência de código.
* **Tópicos abordados:**
  * Alocação dinâmica de memória e ponteiros avançados.
  * Estruturas de dados lineares (Listas encadeadas, Pilhas e Filas).
  * Estruturas não lineares (Árvores e conceitos iniciais de Grafos).
  * Algoritmos de busca e ordenação.
